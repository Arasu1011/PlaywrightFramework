# Playwright Automation Framework (Python)

## Overview

This is a complete Playwright Automation Framework developed using Python, Pytest and Page Object Model (POM).

The framework supports

- UI Automation
- API Testing
- Database Testing
- Data Driven Testing
- HTML Reports
- Logging
- Cross Environment Support
- Git & GitHub

---

# Framework Structure

```
PlaywrightFramework

├── api
├── config
├── database

├── reports
├── screenshots
├── testdata

```

# Execute UI Tests

```
pytest tests -v
```

---

# Execute API Tests

```
pytest api -v
```

---

# Execute Database Tests

```
pytest database -v
```

---

# Generate HTML Report

```
pytest --html=reports/report.html --self-contained-html
```
